
# Light PHP

Inicia el desarrollo de tu aplicación PHP partiendo del modelo de MVC, con los aspectos básicos.




## Instalación

Para usar este proyecto toma en cuenta que debes tener PHP 7.5 o superior.

```bash
  git clone
```
Luego:

```bash
  composer dump-autoload 
```
